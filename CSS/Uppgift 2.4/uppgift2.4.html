Bilder och bakgrunder

I CSS är det vanligt att när bilder stylas så är det enklast att skapa en speciell klass eller id med <div>, därefter stylar man alla bilder som ligger i den taggen. Utan CSS är det förekommade att bilder på webbsidan stylas med img taggen.

Bakgrunder

Bakgrunder i CSS skapas med hjälp av egenskapen background och det kan enkelt bestämmas vilken bakgrund ett speciellt element ska ha eller om hela webbsidan ska ha en viss bakgrund. Med background-image kan det ställas in vilken bakgrundsbild webbsidan ska ha och med background-color kan en färg bestämmas. Exempel visas nedanför: 


body {
    background-image: url(’bild1.jpg’);
} 

h1 { 
background-image: url(’bild2.jpg’);
}

Listor med punkter kan också ha bakgrunder och syntaxen för det ser ut såhär:


.list {
	background-image: url('star.png');
}

Bilder

Med img-taggen kan det infogas bilder på webbsidan och genom att img-taggen blir tilldelas olika egenskaper, kan den kontrollera hur bilden presenteras i dokumentet. Genom att lägga till kantlinjer med border, marginaler som padding ocg margin, hörn som är rundade med border-radius, samt var bilden ska placeras med egenskapen float. Float addresseras med var bilden ska placeras med "right" eller "left", i exemplet visas det hur klassen img, har innehållet float med en riktning till höger och margin med 5 pixlar som innebär att det överiga innehållet ska hålla ett avstånd med 5 pixlar. 


 .img { 
float: right;
margin: 5px;
}

Text i närheten av bilder

När float används kommer texten runt omkring bilden och placeras på andra sidan av elementet. Alltså om float ställs in att vara till höger, placeras text ut till vänster. Om detta inte ska hända, så kan clear hjälpa till att hantera texten tillbaka till att den är under bilden igen. 


p {
clear:left;
}

Clear kan även användas till att styra att vissa taggar som inte ska få flöda som text. Exempel visas nedanför: 


h1 {
clear:left;
}
