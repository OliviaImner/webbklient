Länktillstånd

Om länkar på webbsidan ska styras på ett visst sätt behövs inte bara selektor "a" utan det finns många val att välja mellan. Beroende på vilket tillstånd länken är i, kan det därmed kontrolleras hur utseende på länken ska presenteras på webbsidan.

a:link

Vanligtvis används a:link på webbsidor, denna tagg visar att det finns en länk som muspekaren kan trycka på. a:link är alltså det den visaren för användaren innan en interaktion uppstår mellan elementet och användaren. 


a:link{
color: hotpink; 
}

a:hover

Är hur användaren ska se hur länken tillstånd förändras när muspekaren är över elementet. 


a:hoover{
color: pink;
}

a:active

Selektor a:active är hur tillståndet förändras under den korta perioden användaren faktiskt trycker på länken. Detta är sista tillståndet innan användaren skickas till den vägen till den länkade sidan. 


a:active{
color: #8378F6;
background-color: #FFFFFF;
}

a:visited

Det sista tillståndet är a:visited och är då användaren har besökt länken. 


a:visited{
color: #EA32DB;
}

Cursor

Med cursor bestämms det hur muspekaren ska presenteras när användaren är på webbsidan. Det går att byta utseendet på muspekaren, beroende på vilken del av dokumnetet som användaren drar pekaren över. Exempel på detta visas nedanför:


p{
cursor: pointer;
}
