Anpassning till printer

CSS-konstruktioner för skrivare

@page

Anger layouten på en page box, boxen består av två delar en sid-del och en marginal-del. Det finns också tre olika psudo-selectorer(:first, :left, :right) som kan användas för första sidan, höger och vänster sida. Det går inte att använda sig av px eller pt eftersom skrivare inte förstår detta och använder istället mått som tex cm. Ett exempel kan se ut såhär i kod;


@page {
margin: 3cm;
}
